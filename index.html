React is a javascript Library. The reason that React came into the market is to solve the problem of updating the DOM tree. Whenever we update something the whole DOM tree gets updated that takes too much time and space. Then, React came up with solution that is Virtual DOM. In React, if you want to update some portion of HTML, it doesn't update the whole Virtual DOM ,it only update the requires part of that tree. In React, we divide the websites into various components. React is very famous because it has property of reusable components. React has a concept such as State, Hooks, Redux etc. Initially React component was build in class. Nowadays, React components are functional based.
